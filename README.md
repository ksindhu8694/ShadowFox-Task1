**Overview of the Project**
Project:This project was developed as part of my Machine Learning Internship.  
The objective is to build a regression model that predicts house prices in Boston based on various socio-economic and housing-related features.
Problem Statement:
- Crime rate
- Number of rooms
- Property tax rate
- Distance to employment centers
- Pupil-teacher ratio
- And other housing factors
 Dataset Description:
- Dataset: Boston Housing Dataset
- Number of features: 13
- Target variable: MEDV (House Price)
- Features include:
- CRIM – Crime rate
- RM – Average number of rooms
- AGE – Proportion of old houses
- TAX – Property tax rate
- LSTAT – Percentage of lower status population
Technologies Used:
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  
- VS Code
 Project Workflow
1. Data Loading and Exploration  
2. Data Preprocessing  
3. Feature Selection  
4. Model Training  
5. Model Evaluation  
6. Price Prediction
